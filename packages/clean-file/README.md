## 使用

> npm i -g @smallmonster/clean-file  
> 删除单个文件夹：cleanFile dirName[,dirName1]

## option

### -f

删除文件
> cleanFile -f  fileName[,fileName1]

### -r

需要递归查找几次，默认一次。

> cleanFile -r=number --recursion  arg

### -c

删除前，二次确认

> cleanFile -c --confirm  args -->
